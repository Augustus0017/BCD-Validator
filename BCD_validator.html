BCD Validator

The Binary Encoded Decimal (BCD) is a way of encoding decimal numbers using the binary digits (bits) i.e. 0s and 1s. The digits of a decimal number are each represented by its corresponding binary number with exactly four bits.

The BCD equivalent of the decimal number 12 is 0001 0010 and for the decimal number 93 it is 1001 0011 .

Because the decimal system has ten digits - from 0 to 9, only the binary numbers corresponding to this decimal numbers should be seen in a BCD signal. Any binary number outside the range of 0000 to 1001 should be flagged as invalid BCD code.
